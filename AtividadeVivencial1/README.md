
## controles

- `TAB` : seleciona o próximo objeto
- `W/A/S/D` : transladar o objeto no plano X/Z
- `I/J` : transladar o objeto no eixo Y
- `Z` : iniciar rotação contínua no eixo Z
- `X` : iniciar rotação contínua no eixo X
- `C` : iniciar rotação contínua no eixo Y
- `[` : aumentar escala uniforme
- `]` : diminuir escala uniforme
- `ESC` : fecha a janela

## observações

- O codigo carrega modelos OBJ de `assets/Modelos3D/`
- é importante garantir que os arquivos OBJ existam neste diretório antes de executar o codigo
- use `TAB` para alternar entre os objetos e aplicar transformações ao objeto selecionado
